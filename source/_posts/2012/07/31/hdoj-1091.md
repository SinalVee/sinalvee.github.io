虽然是水题但是依然有需要注意的地方。
`||`与`&&`要弄清楚区别

```c
// hdoj_1091 A+B for Input-Output Practice (III)

#include <stdio.h>
int main(void)
{
	int a, b;
	scanf("%d%d", &a, &b);
	while(a != 0 || b != 0) //这里注意 || 与 && 的区别
	{
		printf("%dn", a + b);
		scanf("%d%d", &a, &b);
	}
	return 0;
}
```

另一种写法

```c
#include <stdio.h>
int main(void)
{
    int a, b;
    while(scanf("%d%d", &a, &b) && (a != 0 || b != 0)) //这里注意 || 与 && 的区别
    {
        printf("%dn", a + b);
    }
    return 0;
}
```
